Assistive Technology Description and Analysis WAVE and NVDA

Developers and scientists proposed various of assistive technologies that can be used in different kinds of devices. All evaluations are based on this technologies in this report. Now I will introduce and analyze two of them.

WAVE

According to the introduction in WebAIM website, “WAVE is a free web accessibility evaluation tool used to aid humans in the web accessibility evaluation process”. WAVE can be used in six different ways. Users can evaluate a website by entering its URL, uploading a file, inputting HTML code directly, installing WAVE toolbar in Firefox, installing WAVE extension in Dreamweaver, or implementing WAVE into web pages, browsers, or other tools.

WAVE can evaluate website in a intuitive way. Users can directly perceived through the indicators and symbols added by WAVE to the original website with “errors, features and alerts” fuction. It is better than providing a complex technical report. With the “structure/order” function, users can easily find out the tab order and structure of the document visually. “Disable styles” function can remove the CSS file from HTML. So users can see the original document avoiding disturbance of styles.

NVDA

NVDA is short for NonVisual Desktop Access, which is an open source screen reader for reading texts displayed in screens to audience, who could be disabled with eyes.

It is highly configurable as well as effectiveness in different scenarios. It also can read not only texts, but also actions and operations from the user. The speech synthesizer built in NVDA supporting over 20 languages, such English, Chinese and so on. The problem of NVDA is that it can only be used in Windows and not compatible with other operation systems.
