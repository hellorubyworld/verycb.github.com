类型转换

// 把字符串转换成整型，10表示的是“123”的进制
parseInt("123", 10) // 123

// 如果要转换二进制数，把10换成2
parseInt("11", 2) // 3

// 转换浮点数
parseFloat("3.14") // 3.14

// “+”号也可以
+ "42" // 42

// 如果要转换的字符串不是数字，则返回“NaN”
parseInt("hello", 10) // NaN

// 判断是否为“NaN”
isNaN(Nan) // true

// 正负无穷
1 / 0 // Infinity
-1 / 0 // -Infinity

// "+"与parseInt()、parseFloat()的区别
+ "10.2abc" // Nan
parseInt("10.2abc" ,10) // 10
parseFloat("10.2abc") // 10.2
